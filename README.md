# Xbox Subscription Sales Dashboard

A dashboard developed in Excel for Web to analyze Xbox subscription sales data.  
This project was created as part of the course:

**TOTVS - Fundamentos de Engenharia de Dados e Machine Learning (TOTVS - Fundamentals of Data Engineering and Machine Learning)**

Challenge: **Criando um Dashboard de Vendas do Xbox com Excel (Creating an Xbox Sales Dashboard with Excel)**

## Dashboard Preview

The dashboard includes:

- Total revenue KPI
- Revenue by subscription plan
- Revenue by subscription type
- Auto renewal analysis

## Business Questions

1. What is the company's total revenue?
2. Which subscription plan generates the highest revenue?
4. Which subscription type generates the most revenue?

## Tools Used

- Microsoft Excel for Web

## Dataset Columns

- Subscriber ID
- Name
- Plan
- Start Date
- Auto Renewal
- Subscription Price
- Subscription Type
- EA Play Season Pass
- Minecraft Season Pass
- Coupon Value
- Total Value
